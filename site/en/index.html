For foreign residents

Filing your tax return in Japan

A short orientation if you're a foreign resident new to Japan's annual personal income tax return process (Kakutei Shinkoku).

Who needs to file?

If you have business income (self-employed, freelance, sole proprietor) or rental income in Japan, you generally need to file a Kakutei Shinkoku between February 16 and March 15 each year, covering the previous calendar year (Jan 1 – Dec 31).

What KAIKEI LOCAL does for you

  • Records your business income/expenses (double-entry bookkeeping)
  • Generates the 青色申告決算書 (Blue Form business statement, ¥650,000 deduction eligible)
  • Calculates Income Tax (所得税) and Consumption Tax (消費税)
  • Outputs the XTX file in NTA's official format (RKO0010 / RSH0010), ready to load on the e-Tax site

What you still need to do yourself

  1. Get a 利用者識別番号 (User Identification Number) from your local tax office. The application form is available in English at the NTA. Once obtained, this is what identifies you to e-Tax.
  2. Submit the XTX via either the NTA's web portal (確定申告書等作成コーナー) or e-Tax software (Windows). You can sign in with My Number Card + smartphone, or with your User ID + password.
  3. Keep paper records of receipts and invoices for 7 years. KAIKEI LOCAL stores receipt photos alongside journal entries to make this easier.

Things to be aware of (foreign-resident specifics)

  • Resident vs non-resident: If you've lived in Japan for 1+ year, you're typically a "resident" for tax purposes. Different rules apply for non-residents — consult a tax professional if unsure.
  • Tax treaties (租税条約): If your home country has a treaty with Japan, you may be eligible for reduced withholding on certain income. KAIKEI LOCAL doesn't currently have specialized fields for treaty claims — you'd add these manually on the e-Tax site.
  • Foreign tax credit (外国税額控除): If you've paid tax abroad on the same income, you may be able to credit it against Japanese tax. Currently this requires manual entry on the e-Tax form after loading the XTX.
  • Name in Katakana: The e-Tax form expects your name in Katakana (フリガナ). KAIKEI LOCAL does not yet auto-convert Romaji to Katakana — you can enter it manually for now.

FAQ

Frequently asked

Is the app really free?
Yes. Bookkeeping, receipts, invoices, blue return forms, income/consumption tax calculation, and e-Tax XTX export are all free. The only paid feature is optional AI receipt OCR.
Is my data really private?
Yes. All data — journal entries, receipt images, invoices — is stored as a SQLite file on your Mac. No internet upload. Phone-to-Mac receipt transfer is over your local Wi-Fi LAN only. The source code is on GitHub so a third party can verify there's no hidden network traffic.
Can I file my Japan tax return entirely with this app?
The app generates the XTX file accepted by Japan's e-Tax system. You then load it on the official e-Tax site or e-Tax software (Windows-only) and submit. Direct submission from a Mac is not possible because the National Tax Agency hasn't implemented Mac native submission yet — this is a Japan-government limitation, not specific to KAIKEI LOCAL.
Does it work for foreign residents in Japan?
Yes. The accounting logic is identical regardless of nationality — the e-Tax XTX format is the same. Two caveats: (1) the UI is currently Japanese only (English planned), and (2) name/address fields expect Japanese-style input (Katakana, Kanji). Romaji-to-Katakana auto-conversion is on the roadmap. See the tax filing guide above for details.
When will English UI be ready?
No firm date yet. Pre-register by email and you'll be notified. The current focus is making the Japanese version stable for the 2025 tax year (filing in early 2026); English UI comes after.
Can I migrate from another accounting app?
You can import journal CSVs exported from freee, MoneyForward, or Yayoi. Migration is not fully automatic, but key data carries over. Opening balances and fixed assets are entered through a guided wizard.
Windows support?
macOS only at the moment. Tauri-based, so Windows is technically possible — we'll prioritize it based on demand.
Can I move data to another Mac?
Yes. From Settings, click once to create a backup ZIP including the DB and receipt images. Restore on the new Mac via the same flow.
Updates for tax law changes?
Yearly updates aligned with Japan's tax-law revisions. New versions are notified in-app on launch; download the latest DMG from GitHub Releases. Updates are free.
macOS warns "developer cannot be verified"
During the beta we don't yet have Apple notarization. Right-click "KAIKEI LOCAL.app" → "Open" while holding Control. See the install guide for details.
