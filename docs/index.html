How it works

3 ステップで、確定申告まで完結

銀行・カードの CSV を入れるところから e-Tax 用 XTX ファイル生成まで、全部 Mac の中で完結。
クラウドに一切データを送りません。

1

銀行 CSV を取り込む

みずほ・三菱UFJ・住信SBI など主要 30+ 行 / カード 11 社の CSV を自動判別。Shift_JIS でも UTF-8 でも、ヘッダの形式を見て勝手に直してくれます。

2

AI が仕訳をつくる

取引内容と過去のルールから勘定科目を推測。「Apple」→ 通信費、「JR」→ 旅費交通費 のように、使うほど精度が上がる学習型エンジン。

3

e-Tax XTX が出力される

所得税申告 (RKO0010) と消費税申告 (RSH0010) の XTX を直接生成。e-Tax マイページの「データ内容確認」で実際にパース・帳票表示まで通ることを確認済み。

Why KAIKEI LOCAL

クラウド会計と何が違うのか

freee や マネーフォワードのようなクラウドサービスとは、設計思想がまるで違います。

SQLite ファイルとしてローカルに保存。クラウドに送信しないので、サービス停止・データ漏洩・規約変更でデータが消えることがありません。

¥0

サブスクなし、ずっと無料

仕訳から青色申告決算書、e-Tax XTX 出力まで全部無料。月額 1,000 円 × 数年というクラウドの累積コストがかかりません。

日本での確定申告にそのまま使えます

外国籍の個人事業主・フリーランスの方も、日本での確定申告 (e-Tax) に対応した会計データを KAIKEI LOCAL で生成できます。利用者識別番号は税務署で英語フォーム提出可能です。

Foreign residents running their own business or freelancing in Japan can use KAIKEI LOCAL to prepare their personal income tax return. The app generates standard e-Tax XTX files (RKO0010 / RSH0010) accepted by the National Tax Agency — same format used by Japanese citizens.

Security

なぜ安心して使えるのか

財務データを預ける以上、信頼の根拠は明示しておくべきだと考えています。

完全オフライン動作

仕訳・領収書・請求書・確定申告データはすべて Mac のローカル SQLite ファイルに保存。クラウドへのバックアップ送信も一切ありません。

データは自分のもの

~/Library/Application Support/dev.kaikei.app/ にファイルとして存在。バックアップの ZIP 化、Finder からの直接アクセス、他 Mac への移行も自由。

日次自動バックアップ

アプリ起動時に DB を snapshots/ に自動保存。直近 7 日分保持。復元時も自動で pre-restore スナップショットを作成するので、失敗時に安全に戻せます。

オープンソース

GitHub で全コードを公開。データの取り扱いも第三者が検証可能。隠れた外部送信はありません。

AI 送信は明示同意制

AI 読み取り機能の画像送信は初回利用時に明示同意ダイアログを表示。送信先・内容・保存ポリシーを開示。同意しない場合はオフライン OCR が自動で動作。

e-Tax 実機で検証済み

令和 7 年分 (2025 年分) の所得税申告 XTX は e-Tax マイページの「データ内容確認」機能でパース・帳票表示まで通過確認済み。

FAQ

よくある質問

基本機能は本当にずっと無料ですか？
はい。仕訳・領収書・請求書・青色申告決算書・所得税/消費税計算・e-Tax 用 XTX 出力まで、すべて無料でご利用いただけます。課金が必要なのは AI 領収書読み取り機能のみです (任意)。
データは本当に外に出ませんか？
はい。全てのデータ（仕訳、領収書画像、請求書）は Mac のローカルフォルダに SQLite ファイルとして保存されます。インターネットへの送信は一切行いません。スマホ連携も同一 Wi-Fi 内の LAN 通信のみです。ソースコードも GitHub で公開しているので、隠れた外部送信がないことを第三者が検証できます。
他の会計ソフトから乗り換えられますか？
freee・マネーフォワード・弥生からエクスポートした CSV (仕訳帳) を取り込むことができます。完全な自動移行ではありませんが、主要なデータは移行可能です。期首残高・固定資産台帳は移行ウィザードで個別入力します。
確定申告はオンラインでできますか？
アプリ内で所得税・消費税を自動計算し、e-Tax 用の XTX ファイルおよび確認用 PDF を出力します。XTX は e-Tax マイページの「作成した申告・申請データの内容を確認」で読み込み可能です。実送信は国税庁の確定申告書等作成コーナーで XTX を読み込むか、e-Tax ソフト (Windows) で組み込み送信する形になります。Mac から直接送信する手段は現時点で国税庁側に存在しません。
Windows には対応していますか？
現在は macOS 専用です。Windows 版は需要に応じて検討します。技術的には Tauri フレームワークを使っているため、Windows 対応は可能です。
別の Mac にデータを移行できますか？
「設定」画面からワンクリックでバックアップ ZIP を作成できます。DB ＋ 領収書画像を丸ごと含みます。新しい Mac でアプリをインストールし、ZIP から復元するだけです。
税制改正に対応しますか？
毎年の税制改正に合わせたアップデート版を提供します。新バージョン公開時はアプリ起動時に通知され、GitHub Releases から最新 DMG を取得できます。アップデート自体は無料です。
AI 読み取りプランはいつでも解約できますか？
はい。Stripe のカスタマーポータルから随時解約可能です。解約後も基本機能は引き続き無料でご利用いただけます。オフラインの Tesseract OCR は解約後も動作します。
初回起動時に「開発元を確認できません」と出ます
現在はベータ配布中のため Apple 公証未取得です。「KAIKEI LOCAL.app」を Control キーを押しながらクリック → 「開く」で起動できます。詳しくは インストールガイド をご覧ください。
対応していない銀行・カードの CSV はどうすれば？
対応リスト外でも汎用パーサが動作します。ヘッダが「日付・摘要・入金・出金・残高」などの標準的な名前なら大丈夫です。うまく読み込めなかった場合は k.nagasawa.pc@gmail.com に CSV のヘッダ行だけお送りいただければ、個別対応を追加します。
