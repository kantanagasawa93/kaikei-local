インストールガイド

KAIKEI LOCAL を Mac にインストールする方法

⬇︎ 最新版をダウンロード (.dmg)

ダウンロード元: GitHub Releases ・ サイズ: 約 30-50MB ・ 必要容量: 100MB 以上

1DMG を開く

ダウンロードした KAIKEI_LOCAL.dmg をダブルクリック。 ボリュームがマウントされて、「KAIKEI LOCAL.app」と Applications フォルダのショートカットが表示されます。

2Applications にドラッグ

「KAIKEI LOCAL.app」を Applications フォルダにドラッグ&ドロップ。 これで /Applications/KAIKEI LOCAL.app にインストールされます。

3初回起動

Launchpad や Applications フォルダから「KAIKEI LOCAL」を起動。

⚠️ 「開発元を確認できません」と表示される場合
現在のベータ配布は Apple 公証を取得していないため、初回起動時に Gatekeeper の警告が出ることがあります。次の手順で開いてください:
  1. Finder で「KAIKEI LOCAL.app」を Control キーを押しながらクリック
  2. メニューから「開く」を選択
  3. ダイアログで再度「開く」をクリック
以降、通常通りダブルクリックで起動できます。正式版では公証済み DMG を配布予定です。

4オンボーディング

初回起動すると簡単なツアー (5 画面) が表示されます。 最後に氏名・屋号・税務署コードなどを入力できますが、あとで設定画面から入力してもOK。

🔒 プライバシー

KAIKEI LOCAL で入力したデータはすべて Mac のローカル (~/Library/Application Support/dev.kaikei.app/) に保存されます。 クラウド送信は一切行いません (AI OCR を明示同意で使う場合のみ、画像が外部サーバで解析されます)。

詳細は プライバシーポリシー をご覧ください。

🆘 困ったとき

  • 起動しない / クラッシュする: メニューバー「ファイル」→「データフォルダを開く」でデータ本体にアクセスできます
  • バックアップ / 復元: アプリの「設定・ヘルプ」画面の「バックアップ」ボタンから
