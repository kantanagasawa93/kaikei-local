変更履歴

PDCA ラウンド単位のリリースノート。最新版から並べています。

v0.3.0 (2026 年 5 月 — Round 2 + Round 3 成果)

受信箱・自動仕訳

  • NEWクイック確定モード — 受信箱の候補カードで「⚡ いますぐ仕訳化」をワンクリックすると、その場で AI OCR → 領収書登録 → 仕訳まで一気通貫
  • NEW失敗の見える化と再試行 — AI OCR で詰まった写真を receipt_failed 状態で残し、エラー理由をカード上に赤バッジで表示。1 件ずつ or 一括で再試行可能
  • NEWOCR 結果の完全保存 — Claude OCR のレスポンスを photo_inbox.claude_result_json に保存し、再仕訳・監査ができる
  • NEW件数バッジ — 受信箱フィルタタブに件数を表示 (未判定 6 / 領収書 4 / 失敗 1 など)
  • NEW未判定に戻す — 「破棄」「違う」「失敗」のカードからワンクリックで未判定に戻せる逆操作
  • NEWOCR テキストの色分けプレビュー — Vision OCR の結果を行単位で「金額」「合計」「日付」「インボイス番号」「店名」と色分け表示し、AI OCR を回す前から「ここまで読めてる」が分かる

取り込みパフォーマンス

  • PERFスマート増分スキャン — iCloud から低画質サムネだけ先に取って文書判定 → 通った写真だけフルサイズダウンロード。1000 枚規模で iCloud 帯域が ~1/20 に
  • FIXマイグレーション自動復旧 — sqlx の checksum mismatch を自動検出し、データベースをバックアップしたうえで _sqlx_migrations をリセット → マイグレーション再適用 (Round 1 で起きたデータ初期化事故を再発防止)

仕訳帳

  • NEW「📷 受信箱」バッジ — 受信箱経由で自動仕訳された行に画像バッジを表示し、領収書一覧に飛べる

配布

  • NEWUniversal Binary 動線 — Apple Silicon (M1〜M4) + Intel Mac の両方に対応
    • Apple Silicon: KAIKEI_LOCAL.dmg または KAIKEI_LOCAL_arm64.dmg
    • Intel Mac: KAIKEI_LOCAL_x64.dmg
  • LP / インストールガイドにアーキ別ダウンロードボタンを追加

開発者向け

  • DEVkaikei --simulate-scan / --db-dump=<table> / --tail-scan-log の CLI 拡張
  • DEVscripts/verify-app.sh smoke で起動・スキャン・スクショまでを自動化
  • DEVGitHub Actions の Verify Round ワークフローで commit ごとに macOS で自動検証
  • DEVscripts/release-precheck.sh でリリース実行前のフル検証 (env / バージョン整合 / 認証 / 空き容量)

v0.2.0 (2026 年 4 月末 — Round 1)

  • FIXVision OCR の致命バグ修正 (VNRequestTextRecognitionLevel の値が逆だった事故)
  • NEW受信箱の精度・表示・操作性を全面立て直し
  • NEWiCloud 写真スキャンの初期版 (PhotoKit + 文書検出フィルタ)
  • NEWAI OCR 送信ログビュアー (透明性のため、送信履歴を確認可能)
  • NEWLaunchAgent によるバックグラウンド定期スキャン

v0.1.0 (初版)

  • ローカル SQLite ベースの会計帳簿
  • 領収書・仕訳・銀行明細・請求書・確定申告 (青色 / 白色) 対応
  • e-Tax XTX 出力
